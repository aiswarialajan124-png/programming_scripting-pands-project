# Iris Dataset Analysis

## Overview
This project involves analysing the Iris dataset using Python. The aim is to explore the dataset, summarise  the variables, and identify patterns between different species of iris flowers.

## Dataset Description
The Iris dataset is introduced by Ronald Fisher in 1936 for evaluating classification methods.

The dataset contains measurements of 150 iris flowers from 3 different species:
 - Sentosa
 - Veriscolour
 - Virginica

Each flower is described using four features:
 - Sepal length (cm)
 - Sepal width (cm)
 - Petal length (cm)
 - Petal width (cm)

These measurements are used to study differences between species and to build classification models.

## Purpose of the Dataset
The Iris dataset is commonly used to test classification algorithms and explore patterns in data.

## Analysis Performed
The following analysis was carried out in this project:
- Calculation of summary statistics for each variable
- Creation of histograms to show distributions
- Generations of scatter plots to examine relationships between variables
- Additional analysis comparing species

## How to Run
1. Open the project folder in VS Code
2. Launch Jupyter Notebook
3. Open 'analysis.ipynb'
4. Run all cells to reproduce analysis
